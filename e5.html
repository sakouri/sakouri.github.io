Projets épreuve E5 : Gestionnaire des interventions opérées par Orange



Objectif :

Ces projets de développement ont pour but de concevoir une application Java Swing et un site web PHP pour la gestion des interventions des techniciens d'Orange. Ils permettront de gérer les techniciens, les clients et les interventions, ainsi que d'administrer la base de données.

Les outils offrent les fonctionnalités suivantes :

- Gestion des techniciens

- Gestion des clients

- Gestion des interventions

- Administration de la base de données

Architecture globale du site web PHP - client léger :

Le site web est composée de trois couches principales (architecture MVC):

- Couche d'accès aux données (Modèle): communication avec la base de données.



- Couche présentation (Vue): affichage de l'interface utilisateur et de la gestion des interactions avec l'utilisateur.



- Couche métier (Contrôleur): logique métier de l'application, traitement des données et des règles métier.
